# API管理后台平台

## 项目简介

这是一个基于Java 17 + Spring Boot 3.x + Vue 3 + Element Plus的现代化管理后台平台，采用RBAC权限模型，支持按钮级权限控制，具备完整的用户管理、角色管理、菜单管理、权限管理、日志管理等功能。

## 技术栈

### 后端技术栈
- **Java 17** - 主要编程语言
- **Spring Boot 3.x** - 核心框架
- **Spring Security** - 安全框架，实现认证和授权
- **MyBatis-Plus** - ORM框架，简化数据库操作
- **MySQL 8.0** - 关系型数据库
- **Redis** - 缓存数据库，用于用户信息、字典数据缓存
- **JWT** - 无状态认证机制
- **Knife4j** - API文档生成工具
- **Druid** - 数据库连接池

### 前端技术栈
- **Vue 3.x** - 前端框架
- **Element Plus** - UI组件库
- **Vue Router** - 路由管理
- **Vuex** - 状态管理
- **Axios** - HTTP客户端
- **Vite** - 构建工具

## 核心功能

### 1. 系统管理
- **用户管理**：用户增删改查、重置密码、用户状态管理
- **角色管理**：角色增删改查、菜单权限分配、数据权限设置
- **菜单管理**：菜单增删改查、按钮权限配置
- **部门管理**：部门树形结构管理
- **字典管理**：系统字典类型和数据管理
- **参数设置**：系统参数配置管理

### 2. 权限管理
- **RBAC权限模型**：用户-角色-菜单-权限的完整权限体系
- **按钮级权限控制**：细粒度的功能权限控制
- **数据权限**：支持全部数据、自定义数据、本部门数据、本部门及以下数据权限
- **动态路由**：根据用户权限动态生成路由菜单

### 3. 日志管理
- **操作日志**：记录用户操作日志，支持导出
- **登录日志**：记录用户登录信息，包括登录时间、IP地址、浏览器信息等
- **在线用户**：实时查看当前在线用户信息

### 4. 系统工具
- **文件上传**：支持单文件和多文件上传
- **Excel导入导出**：支持数据的Excel导入导出功能
- **系统监控**：服务器信息监控、在线用户监控

## 安全特性

### 1. 认证机制
- **JWT Token认证**：无状态认证，支持Token过期和刷新
- **密码加密**：使用BCrypt进行密码加密存储
- **登录验证码**：防止暴力破解
- **会话管理**：Redis存储用户会话信息

### 2. 权限控制
- **动态权限加载**：启动时加载所有权限到缓存
- **权限注解**：使用注解方式控制接口权限
- **路由守卫**：前端路由权限控制
- **按钮权限**：前端按钮显示/隐藏控制

### 3. 安全防护
- **XSS过滤**：防止跨站脚本攻击
- **SQL注入防护**：MyBatis-Plus内置防护
- **CSRF防护**：Token验证机制
- **接口限流**：防止接口被恶意调用

## 快速开始

### 环境要求
- JDK 17+
- Node.js 16+
- MySQL 8.0+
- Redis 6.0+
- Maven 3.6+

### 后端启动
1. 创建数据库：执行 `docs/sql/api_admin_platform.sql`
2. 修改配置：修改 `backend/api-admin/src/main/resources/application.yml` 中的数据库和Redis配置
3. 启动项目：运行 `ApiAdminApplication.java`

### 前端启动
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 访问地址：http://localhost:80
